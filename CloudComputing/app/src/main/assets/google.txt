O Google App Engine é uma plataforma de Computação em nuvem para desenvolver e hospedar aplicações web na infraestrutura do Google. Foi inicialmente lançado como versão preliminar (beta) em Abril de 2008. [1]
O Google App Engine é uma tecnologia no modelo Plataforma como Serviço. Ele virtualiza aplicações em múltiplos servidores, provendo hardware, conectividade, sistema operacional e serviços de software. O Google App Engine pode ser usado gratuitamente até um determinado nível de consumo de recursos. A partir daí, tarifas adicionais são cobradas pelo consumo recursos (armazenamento, banda de rede, ciclos de CPU, etc.) da aplicação.[2]
Google Cloud SQL[editar | editar código-fonte]
Em outubro 2011, o Google previews um zero de manutenção de banco de dados SQL, JDBC e que suporta DB-API. [3] . Este serviço permite criar, configurar e usar bancos de dados relacionais com os aplicativos App Engine. Motor de banco de dados é MySql versão 5.1.59 eo tamanho do banco de dados não pode ser maior do que 10GB [4] .
